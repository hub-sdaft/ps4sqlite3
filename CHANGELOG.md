## 0.1.0

Initial release of Python Snippets for SQLite3!

### Added

- Code snippet for starting quick sqlite3 scripts.
- SQL queries snippets for:
  - INSERT SQLs:
    - INSERT INTO SQLs
    - INSERT INTO SQLs with `?` parameters
  - SELECT SQLs:
    - SELECT WHERE SQLs
    - SELECT WHERE LIKE SQLs
